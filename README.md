# Heist! Repository

This repository is now organized so the root stays focused on the main Vite app and shared project config.

## Structure

- `client/`: main application source code
- `client/src/assets/`: runtime assets imported by the app
- `assets/raw/`: reference media and source art that are not imported at runtime
- `docs/`: product and deployment documentation
- `apps/studyslayer/`: separate secondary app preserved as its own project

## Main App Commands

```bash
npm install
npm run dev
npm run build
```
